# 🍕 Pizza Sales Data Analysis (Interactive Dashboard using MS Excel)
## 🎯 Project Objective
To analyze pizza sales data and build an interactive Excel dashboard that provides insights into revenue performance, customer ordering behavior, product trends, and category-wise sales contribution to support data-driven decision-making.

## Dataset used
- <a href="https://github.com/Akhilas29/Sales-Data-Analysis-SQL/blob/main/Pizza%20Excel%20File.xlsx">Dataset

## Questions (KPIs)
- What is the total revenue generated?
- What is the average order value?
- How many total pizzas were sold?
- What is the total number of orders?
- Which days generate the highest sales?
- What are the peak ordering hours?
- Which pizza category contributes the most revenue?
- Which pizza size drives maximum sales?
- Who are the top 5 best-selling pizzas?
- Which pizzas are the worst performers?

- Dashboard Interaction < a href="https://github.com/Akhilas29/Sales-Data-Analysis-SQL/blob/main/Screenshot%202026-02-23%20183804.png">View Dashboard

## Process
Data Cleaning
- Removed inconsistencies and null values
- Verified data types (date, time, numeric fields)
SQL Analysis
- Used SELECT, JOIN, GROUP BY, and aggregation functions
- Calculated revenue, total orders, and product-level performance
Data Transformation in Excel
- Created Pivot Tables for category, size, and time-based analysis
- Designed calculated metrics (Revenue, AOV, Total Orders)
Dashboard Development
- Built KPI cards
- Added slicers for dynamic filtering
- Created charts for trend analysis and performance comparison

## 📈 Project Insights
- Total Revenue: ₹69,793.30
- Total Orders: 1,845
- Total Pizzas Sold: 4,232
- Average Order Value: ₹37.83
- Weekend sales (Friday & Saturday) are highest
- Classic category contributes maximum revenue
- Large size pizzas generate the highest sales share
- The Pepperoni Pizza is the top-selling product
- The Brie Carre Pizza is the lowest performer

## ✅ Final Conclusion
The analysis revealed clear revenue-driving segments, peak sales periods, and product performance trends. Large-size pizzas and the Classic category are key contributors to overall revenue. Weekend demand significantly impacts total sales. These insights can help optimize inventory planning, marketing strategies, and operational efficiency.
